JavaScript API

Lightpack provides a comprehensive JavaScript library for interactive components. All functions are namespaced under the global Lightpack object.

Auto-Initialization

All components are automatically initialized on DOMContentLoaded:

document.addEventListener('DOMContentLoaded', function () {
  Lightpack.initAll();
});

API Reference

Master Initialization

Lightpack.initAll(root = document)

Initializes all Lightpack components at once.

Parameters:

  • root (Element) - Root element to search within (default: document)

Usage:

// Initialize all components
Lightpack.initAll();

// Initialize within a specific container
const container = document.querySelector('#my-app');
Lightpack.initAll(container);

Component Initializers

Lightpack.initTabs(root = document)

Initializes tab switching functionality.

Markup:

<div class="tabs-group">
  <div class="tabs">
    <button class="tab tab-active">Tab 1</button>
    <button class="tab">Tab 2</button>
  </div>
  <div>Content 1</div>
  <div hidden>Content 2</div>
</div>

Lightpack.initAccordions(root = document)

Initializes accordion behavior (one panel open at a time).

Markup:

<div class="accordion">
  <div class="accordion-header">Section 1</div>
  <div class="accordion-panel">Content 1</div>
  <div class="accordion-header">Section 2</div>
  <div class="accordion-panel">Content 2</div>
</div>

Lightpack.initAlerts(root = document)

Enables alert dismissal via close buttons.

Markup:

<div class="alert alert-success">
  <span>Success message</span>
  <button class="alert-close">&times;</button>
</div>

Lightpack.initModals(root = document)

Initializes modal dialogs with backdrop and keyboard support.

Markup:

<button data-modal-open="myModal">Open Modal</button>

<div class="modal-backdrop" data-modal="myModal" style="display:none;">
  <div class="modal">
    <div class="modal-header">
      <h3 class="modal-title">Title</h3>
      <button class="modal-close" data-modal-close>&times;</button>
    </div>
    <div class="modal-body">Content</div>
    <div class="modal-footer">
      <button data-modal-cancel>Cancel</button>
    </div>
  </div>
</div>

Features:

  • Click backdrop to close
  • Press ESC to close
  • Multiple modals supported
  • Body scroll lock when open

Lightpack.initDrawers(root = document)

Initializes slide-in drawer panels.

Markup:

<button data-drawer-open="myDrawer">Open Drawer</button>

<div class="drawer" id="myDrawer">
  <div class="drawer-backdrop"></div>
  <div class="drawer-panel">
    <button data-drawer-close>&times;</button>
    <p>Drawer content</p>
  </div>
</div>

Features:

  • Left or right side (.drawer-left)
  • Click backdrop to close
  • Press ESC to close
  • Body scroll lock when open

Lightpack.initCollapsibles(root = document)

Initializes collapsible/expandable content sections.

Markup:

<!-- Multi-collapse (default) -->
<div class="collapsible">
  <div class="collapse-item">
    <button class="collapse-toggle">Section 1</button>
    <div class="collapse-content">Content 1</div>
  </div>
</div>

<!-- Accordion mode (one open at a time) -->
<div class="collapsible" data-collapse="accordion">
  <div class="collapse-item">
    <button class="collapse-toggle">Section 1</button>
    <div class="collapse-content">Content 1</div>
  </div>
</div>

Modes:

  • data-collapse="multi" - Multiple sections can be open (default)
  • data-collapse="accordion" - Only one section open at a time

Indicators:

<button class="collapse-toggle">
  <span>Section</span>
  <span class="collapse-indicator" data-open="-" data-closed="+">+</span>
</button>

Lightpack.initDropdowns(root = document)

Initializes smart dropdown menus with viewport edge detection.

Markup:

<div class="dropdown">
  <button class="btn btn-primary">Dropdown</button>
  <div class="dropdown-menu">
    <button class="dropdown-item">Action 1</button>
    <button class="dropdown-item">Action 2</button>
  </div>
</div>

Features:

  • Auto-positions to prevent viewport overflow
  • Fixed positioning for proper layering
  • Keyboard accessible

Lightpack.initPasswordToggles(root = document)

Enables password visibility toggle buttons.

Markup:

<input type="password" id="pwd">
<button type="button" class="toggle-password">
  <i class="fa fa-eye"></i>
</button>

Features:

  • Toggles between type="password" and type="text"
  • Adds .showing class for icon swapping
  • Updates aria-label

Toast Notifications

Lightpack.showToast(options)

Displays a toast notification.

Parameters:

  • options (Object):
    • message (string) - Toast content
    • type (string) - 'success', 'danger', 'warning', 'info' (optional)
    • sticky (boolean) - If true, doesn't auto-dismiss (optional, default: false)
    • position (string) - 'bottom-right', 'bottom-left', 'top-right', 'top-left' (optional, default: 'bottom-right')

Usage:

// Simple toast
Lightpack.showToast({
  message: 'Operation successful!',
  type: 'success'
});

// Sticky toast (manual close only)
Lightpack.showToast({
  message: 'Important notice',
  type: 'warning',
  sticky: true,
  position: 'top-right'
});

Lightpack.removeToast(id, position)

Manually removes a toast by ID.

Parameters:

  • id (number) - Toast ID
  • position (string) - Toast position

Theme Management

Lightpack.setTheme(theme)

Sets the application theme.

Parameters:

  • theme (string) - 'theme-dark' or '' (light)

Usage:

// Enable dark mode
Lightpack.setTheme('theme-dark');

// Enable light mode
Lightpack.setTheme('');

Features:

  • Adds/removes .theme-dark class on <body>
  • Persists preference in localStorage

Lightpack.toggleTheme()

Toggles between light and dark themes.

Usage:

// Toggle theme
Lightpack.toggleTheme();
<button onclick="Lightpack.toggleTheme()">Toggle Theme</button>

Lightpack.initTheme()

Initializes theme from localStorage on page load.

Usage:

// Automatically called by initAll()
// Or call manually:
Lightpack.initTheme();

Utility Functions

Lightpack.toggleClasses(targetSelector, classNames)

Toggles one or more classes on target elements.

Parameters:

  • targetSelector (string) - CSS selector for target element(s)
  • classNames (string|array) - Class name(s) to toggle

Usage:

// Toggle single class
Lightpack.toggleClasses('.sidebar', 'open');

// Toggle multiple classes
Lightpack.toggleClasses('.menu', ['active', 'visible']);

// Toggle on multiple elements
Lightpack.toggleClasses('.item', 'selected');

Lightpack.generateToc(targetSelector, options)

Generates a table of contents from page headings.

Parameters:

  • targetSelector (string) - CSS selector for ToC container
  • options (Object):
    • levels (array|string) - Heading levels to include (default: [2,3,4,5,6])
    • scrollOffset (number) - Offset for fixed headers in pixels (default: 0)
    • activeClass (string) - Class for active section link (default: 'toc-active')

Usage:

// Basic usage
Lightpack.generateToc('.toc');

// Custom options
Lightpack.generateToc('.toc', {
  levels: [2, 3],           // Only h2 and h3
  scrollOffset: 64,         // 64px offset for fixed header
  activeClass: 'toc-active' // Custom active class
});

// String format for levels
Lightpack.generateToc('.toc', {
  levels: 'h2,h3,h4'
});

Features:

  • Auto-generates IDs for headings without them
  • Nested list structure
  • Smooth scrolling
  • Active section highlighting
  • Respects scroll offset

Markup:

<nav class="toc"></nav>

<script>
  Lightpack.generateToc('.toc', {
    levels: [2, 3],
    scrollOffset: 64
  });
</script>

Complete Example

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Lightpack JS Example</title>
  <link rel="stylesheet" href="lightpack.min.css">
</head>
<body>
  <!-- Components auto-initialize -->
  <button data-modal-open="demo">Open Modal</button>
  <button onclick="Lightpack.showToast({ message: 'Hello!', type: 'success' })">
    Show Toast
  </button>
  <button onclick="Lightpack.toggleTheme()">Toggle Theme</button>

  <!-- Modal -->
  <div class="modal-backdrop" data-modal="demo" style="display:none;">
    <div class="modal">
      <div class="modal-header">
        <h3 class="modal-title">Demo Modal</h3>
        <button class="modal-close" data-modal-close>&times;</button>
      </div>
      <div class="modal-body">Modal content here</div>
    </div>
  </div>

  <script src="lightpack.min.js"></script>
  <!-- Components auto-initialize on DOMContentLoaded -->
</body>
</html>

Manual Initialization

If you dynamically add components after page load:

// Add new content
const container = document.querySelector('#dynamic-content');
container.innerHTML = `
  <div class="tabs-group">
    <div class="tabs">
      <button class="tab tab-active">Tab 1</button>
      <button class="tab">Tab 2</button>
    </div>
    <div>Content 1</div>
    <div hidden>Content 2</div>
  </div>
`;

// Re-initialize within that container
Lightpack.initTabs(container);
